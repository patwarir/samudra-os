# Samudra OS

A prototype RISC-V operating system written in Rust.

## Specifications

Supports:

* RISC-V (ISA: `riscv64gcv_zba_zbb_zbc_zbs_zfh`, ABI: `lp64d`)

## Dependencies

* QEMU
* Rust bare-metal toolchain
* C bare-metal toolchain

## Quickstart

Run `./scripts/dev.sh`
